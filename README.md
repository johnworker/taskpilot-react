## 快速開始（Quick Start）

### 1.安裝與啟動開發伺服器

# 進入專案資料夾
cd taskpilot-react

# 安裝相依
npm ci

# 啟動開發伺服器（預設 http://localhost:5173 或依你的 Vite 設定）
npm run dev
